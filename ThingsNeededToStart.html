Things needed to start

This are the things you need to start playing a guitar

  • A Guitar: Well that was obvious!!
  • A cable and an amp: This is only needed if you are going to use a electric guitar.
  • A Guitar Pick: Well this is optional but still recommended
  • A Capo: This is also optional but having it will allow you to play more songs
  • A Guitar Tuner/Tuner App: This is a 100% a essential, but you do not have to buy the tuner.
    Using a tuner app is mor beginner-friendly
